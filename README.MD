```bash
from nurec import Nurec
```

```python
recognized_data = [{<JSON_PARAMS>}, {...}]

result = Nurec(recognized_data)
```

### Methods

#### accuracy

`result.accuracy()`

returns accuracy like `100.00` or worse

#### details

`result.details()`

```bash
[key] file
distance: Number
expected:
String
received:
String
```

#### total_distance

returns total Levenshtein distance

`result.total_distance()`

Number
